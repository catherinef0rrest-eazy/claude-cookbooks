## Usage Instructions

To test with any of these inputs:

1. Copy the desired test input section (including the email format)
2. Save it to a file (e.g., `vena_test.txt`)
3. Run: `python test_skill.py --email-file vena_test.txt`

Or use the default Vena Solutions test by running:
```bash
python test_skill.py
```

For non-interactive testing (auto-approve Stage 2):
```bash
python test_skill.py --auto-approve
```

For Stage 1 only (no JSON conversion):
```bash
python test_skill.py --skip-stage2
```
